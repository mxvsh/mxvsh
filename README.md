## Wave

<img src="https://github.com/mxvsh/wave/raw/main/wave.svg" width="90">

Lightweight, native macOS dictation app built on whisper.cpp.

**Features**

- Global shortcut to start dictation instantly from anywhere in macOS
- Local transcription powered by Whisper for privacy and offline use
- Minimal, distraction-free interface designed for fast voice input
- Automatically pastes transcribed text into the active application

[View Repository →](https://github.com/mxvsh/wave)
